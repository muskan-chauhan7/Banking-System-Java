# 💳 Banking Information System (Core Java)

## 📌 Project Description
The Banking Information System is a console-based application developed using Core Java that simulates basic banking operations. It allows users to register, log in securely, and perform transactions like deposit, withdrawal, and fund transfer.

---

## 🚀 Features
- User Registration with unique account number  
- Secure Login using password  
- Deposit & Withdrawal functionality  
- Fund Transfer between accounts  
- Transaction History (Account Statement)  
- Error Handling (invalid login, insufficient balance, etc.)  

---

## 🛠️ Technologies Used
- Core Java  
- OOP Concepts (Classes & Objects)  
- ArrayList (Data Storage)  
- Scanner (User Input)

## ▶️ How to Run
1. Open project in VS Code  
2. Open Terminal  
3. Compile:javac *.java
4. Run:java Main
